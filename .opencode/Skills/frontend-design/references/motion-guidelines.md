# Motion Guidelines

Use this file when the interface needs transitions, motion, or interaction polish.

## Motion Priorities

- reinforce hierarchy
- support orientation
- improve perceived quality
- avoid decorative noise

## Good Uses

- page and section entry
- staggered reveal for structured content
- drawer and modal transitions
- state changes that clarify interaction

## Avoid

- constant motion everywhere
- aggressive animation on dense work surfaces
- ornamental effects without informational value
- excessive micro-interactions

## Intensity Rules

- marketing surfaces can tolerate stronger motion
- dashboards and tools should prefer controlled motion
- always respect reduced-motion expectations
